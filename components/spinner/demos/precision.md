---
title: 精度
order: 1.3
---

给组件添加`precision`属性可以指定输入值的小数位数，当指定为`0`时，则只能输入整数

```vdt
import Spinner from 'kpc/components/spinner';

<div>
    <Spinner v-model="value1"
        max={{ 10 }}
        min={{ 0 }}
        precision={{ 0 }}
    />
    <Spinner v-model="value2"
        max={{ 10 }}
        min={{ -10 }} 
        step={{ 0.1 }}
        precision={{ 1 }}
    />
</div>
```

    margin-right 20px
